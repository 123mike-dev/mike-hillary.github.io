## Project X: Poster Design for Charity Event

### Overview
This is a poster designed for a charity event aimed at raising awareness for community health initiatives. The goal was to create a visually appealing and professional poster that communicates the message clearly.

### Tools Used
- Adobe Illustrator
- Adobe Photoshop
- CorelDraw (optional)

### Features
- Eye-catching and modern visual layout
- High-quality print-ready file
- Consistent color scheme and typography
- Suitable for both digital and print formats
